A Flutter-based mobile application designed to streamline private security service management through a centralized and user-friendly platform.

---

## 📖 Overview

The Private Security Management System was developed to improve the administration and coordination of private security operations. The application provides an intuitive interface for managing security personnel, clients, services, shifts, incidents, and operational reports.

This project was developed as part of my Computer Engineering studies to strengthen my software development, mobile application design, and system architecture skills.

---

## ✨ Features

- 🔐 User authentication
- 👮 Security personnel management
- 👥 Client management
- 📅 Shift scheduling
- 🛡️ Security service management
- 🚨 Incident reporting
- 📝 Service logs
- 📊 Reports and statistics
- 🔔 Notifications
- 📋 Personnel evaluations
